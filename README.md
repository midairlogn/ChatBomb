# ChatBomb
这是`QQ/TIM/微信/Discord`等聊天软件的刷屏工具。    

## 效果截图
![screenshot](https://raw.githubusercontent.com/midairlogn/ChatBomb/main/screenshot.png)

## 适用系统   
- Windows   

## 操作步骤    
1.	运行`ChatBomb.exe`( *可能需要管理员权限，正常情况下不需要* )   
2.	按照提示语操作即可！   

## 自定义发消息间隔（单位：毫秒）     
默认50毫秒。     
修改`int sleeptime = 50;`中的数值。调整发消息鼠标键盘操作的间隔，发消息间隔是其4倍，建议使用50-100。    
之后重新编译即可。
